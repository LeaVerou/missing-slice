CSS conic-gradient() polyfill

Features

  • Color stop normalization
  • Follows background-size (pixels and percentages)
  • Supports double position syntax (two positions for the same color stop)

Limitations

  • No center support
  • No calc() support
  • No length support in color stops
  • No midpoint color stops (color stops with no color)
  • No offsets < 0 or > 100%
  • No repeating gradients
  • Color interpolation happens without premultiplied alpha.
